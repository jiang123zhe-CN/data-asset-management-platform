# 数据资产管理平台

面向金融机构的数据资产分类分级管理平台，内置**《金融信息服务数据分类分级指南》**（国信办通字〔2026〕2号）合规引擎。

## 核心功能

### 金融合规分类分级
- **67 类三级标准分类**：业务数据 / 用户数据 / 企业数据 → 9 类二级 → 67 类三级，完整覆盖金融信息服务数据范围
- **18 格矩阵判定**：影响对象（国家安全/经济运行/社会秩序/公共利益/组织权益/个人权益）× 危害程度（特别严重/严重/一般）→ 数据级别（核心/重要/敏感一般/常规一般）
- **三大合规原则**：就高从严（表级继承）、级别不可降（≥参考最低级别）、30% 变化阈值（核心/重要数据动态追踪）
- **5 层匹配引擎**：精确关键词 → 产品信号 → 业务域映射 → 兜底推断 → 遍历评分，动态置信度计算

### 数据打标
- 一键执行：金融合规分类（矩阵判定）/ AI 辅助 / 全流水线
- 人工修正：下拉 67 类标准分类 + 四级合规级别
- 结果导出 Excel（含分类路径 + 级别 + 方法 + 置信度）

### 合规输出（P2）
- **分类分级清单导出**：符合指南 §6.4 标准格式，含三级分类路径、参考级别、实际级别、分级理由
- **30% 阈值快照**：创建基线 → 自动对比变化率 → 超阈值告警 → 历史追溯
- **数据打标结果导出**：全部字段的完整打标信息

### 其他
- 字段管理：CRUD + Excel 导入导出 + 自动合规分类
- 映射管理：字段↔资产目录映射，AI 自动映射（仅映射到最深层级）
- 人工复核：异常检测 + AI 建议审核工作流
- 可视化：ECharts 力导向图 + 统计报告

## 技术栈

| 层 | 技术 |
|---|------|
| 后端 | FastAPI (Python 3.12) + SQLAlchemy + JWT |
| 前端 | React 19 + Ant Design 6 + Vite |
| AI | 通义千问 (DashScope API) |
| 数据库 | SQLite (开发) / PostgreSQL (生产) |
| 部署 | 阿里云 ECS 2C2G + Nginx + Systemd |

## 快速开始

### 本地开发
```bash
# 后端
cd backend
pip install -r requirements.txt
python seed.py                                          # 初始化数据
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# 前端
cd frontend
npm install
npm run dev                                             # http://localhost:5173
```

默认管理员：`admin` / `admin123`

### 生产部署
```bash
# 在阿里云服务器上以 root 执行
bash deploy/deploy.sh
```

部署后访问：`http://服务器IP/dam/`

- [x] 67 类三级标准分类
